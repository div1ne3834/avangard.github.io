# Сайт об авангарде в искусстве

Образовательный веб-сайт, посвящённый авангардным течениям в искусстве XX века.

## Структура проекта

- `index.html` - главная страница
- `movements.html` - подробная информация о течениях
- `styles.css` - стили сайта
- `script.js` - интерактивность

## Особенности

- Адаптивный дизайн
- Плавная анимация
- Современный интерфейс в духе авангарда
- Информация о основных течениях: кубизм, футуризм, супрематизм, дадаизм

## Запуск

Откройте `index.html` в браузере.

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, анимации)
- Vanilla JavaScript
